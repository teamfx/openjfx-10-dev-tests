Testing system clipboard contents after selecting "Copy Image to Clipboard" item in context menu

  1. Wait for page to load.
  2. Bold text "Copy PNG Image Type" and below that an image should appear. Right-click on the image. Context menu should appear. Click on "Copy Image to Clipboard" menu item.
  3. After step 2 click on "Verify Clipboard" button. The button text should display Success in green after the click. If the button text displays Failure, it signifies failure of test.
  4. Click on "Next Image" button to proceed to next test.
  5. Repeat steps 2 and 3 for jpg and gif image formats.
  6. After all the images are loaded, "Test Concluded. Click to start over again!" button will be displayed. Click on this button if you want to retest.
The test passes if all the steps pass.
