Test for verifying clipboard contents on drag of text and image

  1. Wait for the test page to load.
  2. Select the "SELECT ME AND DRAG", drag and drop it to the label on the right with text "Drag Text Here". On drag enter of the label, Success message in green should get displayed.
  3. In section "DRAG IMAGE" note the type of image(png,jpg,gif) in the header. Select the image, drag and drop it to the label on the right with text "Drag Image Here". On drag enter of the label, Success message in green should get displayed.
  4. Click on "Next Image" to see a different format image get displayed
  5. Repeat step 3 and 4
  6. After all the image formats are tested, the "Next Image" button will change to "Test Concluded. Click to start over again!"
  7. Click the "Next Image" button again if you want to retest
The test passes if every step passes.
